Plast Pack

by Plastix

A PVP customized version of the default Minecraft resource pack made for the Overcast Network!

Download Releases

About

The aim of the Plast Pack is to create a beautiful and vibrant PVP resource pack. It includes many custom and tweaked textures that add to the game, not redesign it. Minecraft is beautiful, let's see to that!

Notable Features

  • Clear GUI
  • White/Black GUI
  • Skinny tools
  • Custom HD Font
  • Clear water

Other Features

The Plast Pack supports the following Optifine/MCPatcher features. They are not required for the pack to work, but are highly recommended.

  • Connected Textures
    • Clear Glass
    • Random textures
  • Custom Colors
    • Full bright
    • Vivid colors
  • Natural Textures
    • Nicer texture tiling
